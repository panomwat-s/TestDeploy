1. Backend (Flask)

ติดตั้ง Python https://www.python.org/downloads/ เวอร์ชั่นล่าสุด

2.เข้าโฟลเดอร์ backend:

cd backend
python -m venv venv
.\venv\Scripts\activate

3.ติดตั้ง dependency:pip install -r requirements.txt

4.python run.py


5.Frontend (React + Vite)

ติดตั้ง Node.js 
node -v
npm -v

6.เข้าโฟลเดอร์ frontend:

cd frontend
npm install

7.npm run dev




สร้าง Role Admin เข้าโฟลเดอร์  backend
cd backend
python -m venv venv
.\venv\Scripts\activate

rom app import db, bcrypt
from app.models import User

admin = User(
    username="admin",
    email="admin@crm.com",
    role="Admin",
    password_hash=bcrypt.generate_password_hash("admin123").decode("utf-8"),
    is_temp_password=False
)

db.session.add(admin)
